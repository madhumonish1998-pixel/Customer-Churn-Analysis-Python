# 📊 Customer Churn Analysis – Python

## Project Overview
This project performs End-to-End Exploratory Data Analysis (EDA) on customer churn data to identify key drivers influencing customer attrition.

The analysis focuses on customer behavior patterns, service usage, tenure, and revenue contribution to provide actionable retention insights.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Business Objectives
1. Calculate overall churn rate
2. Identify high-risk customer segments
3. Analyze churn by tenure, contract type, and payment method
4. Study revenue impact due to churn
5. Provide retention strategy recommendations

---

## Key Analysis Performed
- Data Cleaning & Missing Value Treatment
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Customer Segmentation
- Churn Rate Calculation
- Visual Trend Analysis

---

## Key Insights
- Customers with month-to-month contracts show higher churn rate
- Lower tenure customers are more likely to churn
- High monthly charges correlate with increased churn probability
- Targeted retention strategies can reduce churn risk
